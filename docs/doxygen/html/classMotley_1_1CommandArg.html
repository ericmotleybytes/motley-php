Set/unset the requirement that the argument specify an existent file.

If set to true, the requirement that a file not exist is set to false.

Parameters
$flag- true if argument is a file specification, else false. If false, the $exist parameter is ignored even if specified.
$exist- if not specified it means file existence does not mater, true means file must exist, false means file must not exist.
