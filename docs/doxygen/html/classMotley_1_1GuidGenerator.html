Detailed Description

Generate unique 128 bit GUIDs/UUIDs as 32 hex characters.

This class includes routines to generate GUIDs (Globally Unique IDentifiers), also called UUIDs (Universally Unique IDentifiers) in accordance with UUID V4 standards (see https://en.wikipedia.org/wiki/Universally_unique_identifier).
